This Project implements the required assignment - normalizing datasets of various formats to a standard format, given both input and output as CSV files.

The file Program.cs implements a C# Class, which contains the required static function
    StandardizeStatement(string inputFile, string outputFile)
When run directly through the Main() function, the user is expected to provide the absolute paths of the-
1. Input CSV File
2. Directory of Desired Output File

The SampleInput Data and the corresponding output files are given in the "InputData" and "OutputData" subdirectories respectively.
